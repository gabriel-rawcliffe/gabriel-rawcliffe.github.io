CSS inline vs inline-block vs block

This post aims to explain the difference between the inline, inline-block, and block elements, and how their use will change how your elements are displayed

In the inline display options, elements do not adhere to width and height values; instead they are all made the same size, and appear horizontally to fit the available space.

Elements displayed as Inline-block can have specified width and height values; the blocks will display horizontally as described in the .css styling.

If the display option Block is used, elements can have specified width and height values, but the blocks will display vertically down the page. If no values are given for width and height, the elements will be sized to take up the available space.
